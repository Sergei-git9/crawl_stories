  ## **5. Логика отображения эпизодов**

### **5.1. Если эпизод опубликован сразу**

* Сразу отображается
* В списке эпизодов появляется метка **«Новый»**
* Когда выходит следующий эпизод > метка «Новый» переносится на следующий

### **5.2. Если эпизод с датой**

* Пока дата не наступила > эпизод:

  * закрыт
  * пометка: **«Недоступный»**

Когда дата + время наступают:

* Эпизод автоматически открывается
* Выходит в ленту

### **5.3. Если эпизод скрыт**

* На сайте не показывается
* В админке виден, пока не изменят статус

---

## **5.4. Итоговая логика поведения истории и эпизодов**

| Условие                       | Что происходит                                  |
| ----------------------------- | ----------------------------------------------- |
| История закрыта               | Показывается только как «Анонс», нельзя открыть |
| История открыта               | Можно открыть описание                          |
| История открыта + есть эпизод | Переход из «Анонс» > «Новинка»                  |
| 2 эпизода вышли               | Метка «Новинка» исчезает                        |
| Эпизод с датой                | Закрыт до времени                               |
| Эпизод опубликован сразу      | Метка «Новый»                                   |
| Следующий эпизод вышел        | Метка «Новый» переходит на новый эпизод         |
| Эпизод скрыт                  | Не показывается                                 |
